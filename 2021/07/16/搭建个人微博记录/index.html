搭建个人微博记录

准备环境

准备node和git环境

安装NodeJS和git

安装后在命令行中验证是否成功

安装Hexo

在命令行输入执行：npm install -g hexo-cli

安装完成后，再执行下列命令，Hexo 将会在指定文件夹中新建所需要的文件

hexo init myBlog

cd myBlog

npm install

可以看到文件夹中如下

运行hexo s

网页输入http://localhost:4000

注册Github

打开仓库创建一个 index.html 文件，写个 Hello World

访问https://lnlove0519.github.io就可以看到站点了

配置SSH key

命令行输入cd ~/.ssh

全局配置本地账户

git config --global user.name "LNLOVE0519"

git config --global user.email "1761723451@qq.com"

开始生成密钥 SSH key

ssh-keygen -t rsa -C '1761723451@qq.com'

通过C:\Users\Lance/.ssh/id_rsa.pub文件内容，获取SSH key

登录 Github 上添加刚刚生成的SSH key

首次使用还需要确认并添加主机到本机SSH可信列表。

ssh -T git@github.com

若返回 Hi xxx! You’ve successfully authenticated, but GitHub does not provide shell access. 内容，则证明添加成功。

部署到Github

打开项目根目录下的 _config.yml 配置文件配置参数

安装一个部署插件 hexo-deployer-git

npm install hexo-deployer-git --save

hexo g -d

g 是 generate 缩写，d 是 deploy 缩写

在浏览器访问网址： https://LNLOVE0519.github.io 就可以看到博客

写作

新建文章hexo new '文章标题'

hexo clean

hexo g -d

上传成功
